# Distroless FOSS Telegram Bot: "The Gatekeeper"

**Цель:** Максимально тупой, легковесный и автономный бот для защиты чатов от спам-ботов.
**Стек:** Go + Altcha + Podman/Docker Scratch.
**Принцип:** YAGNI (You Ain't Gonna Need It) — никакого мусора, TLS, вайтлистов и динамики.

---

## 🏗️ Архитектура

- **Размер образа:** ~12-15 МБ
- **Зависимости:** 0 внешних сервисов (Altcha работает локально)
- **Приватность:** 100% (Весь трафик у вас)
- **Сложность:** Минимальная

## 🚀 Быстрый старт

### 1. Создание Telegram бота

1. Найдите [@BotFather](https://t.me/botfather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям, получите **BOT_TOKEN**

### 2. Сборка образа

```bash
podman build -t bot-guard .
# или с Docker:
# docker build -t bot-guard .
```

### 3. Генерация секретного ключа

```bash
# Сгенерируйте случайную строку для HMAC_KEY
openssl rand -base64 32
```

### 4. Запуск

```bash
podman run -d \
  --name bot-guard \
  -p 8080:8080 \
  -e BOT_TOKEN="123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11" \
  -e BASE_URL="http://YOUR_PUBLIC_IP:8080" \
  -e HMAC_KEY="super-secret-random-string" \
  bot-guard
```

### 5. Добавление бота в чат

1. Добавьте бота в чат как администратора
2. Дайте права: **Ban Users** и **Delete Messages**
3. Все! Бот будет автоматически проверять новых участников

## 🔧 Переменные окружения

| Переменная | Обязательна | Описание |
|-----------|-------------|----------|
| `BOT_TOKEN` | ✅ | Токен Telegram бота от @BotFather |
| `BASE_URL` | ✅ | Публичный URL вашего сервера (http://IP:8080) |
| `HMAC_KEY` | ✅ | Случайная строка для подписи Altcha |
| `PORT` | ❌ | Порт HTTP сервера (по умолчанию: 8080) |

## 🛡️ Как это работает

1. **Новый пользователь вступает в чат**
   - Бот автоматически мьютит его (запрещает писать)
   - Отправляет сообщение с кнопкой "Я не робот 🤖"

2. **Пользователь нажимает кнопку**
   - Открывается веб-страница с POW-задачей (Altcha)
   - Браузер решает небольшую вычислительную задачу

3. **Проверка решена**
   - Бот проверяет решение локально (без внешних сервисов)
   - Успешно ⇒ разбанивает пользователя
   - Неуспешно ⇒ оставляет в мьюте

## 🔒 Безопасность

- **Никаких внешних API:** вся логика работает локально
- **Distroless контейнер:** минимальная поверхность атаки
- **Proof of Work:** защита от автоматических атак
- **HMAC подпись:** защита от подделки решений

## 🐛 Поиск проблем

### Бот не стартует
```bash
# Проверьте переменные окружения
podman inspect bot-guard | grep Env

# Проверьте токен
curl -H "Authorization: Bot YOUR_TOKEN" https://api.telegram.org/botYOUR_TOKEN/getMe
```

### Пользователи не получают сообщения
- Убедитесь что бот админ с правами на бан
- Проверьте что `BASE_URL` доступ извне
- Откройте порт в файрволе

### Капча не работает
- Проверьте консоль браузера на ошибки
- Убедитесь что `HMAC_KEY` одинаковый для всех инстансов
